事前認識

想定環境

  • モード：TVモード
    • 故に、タッチは不使用
      • ほとんどの走者がTVモードでプレイしているが、一方で、例えば Demolition14さんによる1h20m39s（現時点で世界3位）のプレイ はテーブルモードによるもの
  • コントローラー：Switch Proコントローラー
    • くるくるハンドルは、（Joy-Conを振り回してではなく）Rスティックで操作
  • ボタン
    • ダッシュ：Lスティック & Y ボタン
    • ひきぬく／なげる：B ボタン
    • ポインターでタッチ：コントローラーに内蔵されたセンサーによるポインティング & ZRボタン
      • この操作により敵やギミックを止める・お化け系を透明化させる・ブロックを破壊するなどのことを、以下「 シビシビ 」と呼称
        • 由来は 「いい大人達」による実況動画
    • ポインターをもどす： + ボタン
      • 各コースについて、 コースに入りコースの全景が見え始めた段階でのコントローラーの位置 = ゲーム画面中央 と初期設定され
      • ポインターをもどすボタンを押すと、その時点でのコントローラーの位置 = ゲーム画面中央に
    • オプション：-ボタン
      • 「やりなおす」のに使用

タイマーのタイミング

  • スタート：1-Prologueでコントローラー操作によりノピオ隊長が動きだした瞬間
    • そのため、例えば-3秒からタイマーをスタートさせて0になった瞬間以降にキノピオ隊長を動かし始めるといった工夫が必要
    • タイマースタート前にゲーム内の設定を変更することは可
  • ストップ：3-28でパワースターに触れた瞬間
  • セグメント：明確な決まりはないが、多くの走者が1コース = 1セグメントと設定
    • ただし、各エピソードのプロローグと最初のコースは、ひとまとめにしている走者が多め
      • SLDCはこれら2つを別々のセグメントとして設定
  • セグメント間のスプリット箇所：明確な決まりはないが、各コースで「COURSE CLEAR!」の文字が最も大きくなった瞬間（≒クリア時のSEが「ジャン」と終止する瞬間）でセグメントを区切る走者が多め
    • ただし、3-28は上記のタイマーストップタイミングに準拠

操作キャラの基礎動作

  • ダッシュ状態で正反対の方向に折り返そうとすると、ブレーキがかかりながら多少滑った後、振り返り

    • 回り込むように旋回するか、ダッシュを解除して正反対の方向に折り返すと速い
      • ただし、ツルハシ保有状態ではダッシュしたまま正反対の方向に折り返してもロスなし

  • はしごの長さにもよるが、 はしごを降りるよりもはしごの付近から落ちるほうが速い

    • はしごが長いほど降りるときと落ちる時とのタイム差は増加

  • 次のコースに入るための本のページ送りの速さも多少は重要

    • 一方で、 ページをめくりすぎてしまうことは1秒程度のロス

  • 高低差が大きいところを落下する際は、 ダッシュを解除すると尻もちをついてしまうことを回避できる場合があり、また、尻もちをつかない高低差であっても心なしか落下後の動き出しが少し速くなる 印象

    • ツルハシ保有状態でも同様

  • ヒキヌケ草やモグラを引き抜くなどしてダイヤが現れた場合は、 引き抜いた位置から動かなくてもダイヤを取得可能

  • シビシビ状態は、ZRボタンを離してからも多少継続

    • 周期的に動くギミックを一時的にシビシビ→解除して利用（いわゆる周期調整）する場合は、ZRボタンを離してからのシビシビ継続時間も考慮する 必要あり

「やりなおす」の仕様

  • コース内で、オプションから「やりなおす」を選択すると、コースに入った直後の状態に
    • ただし、 やりなおす前に取得したダイヤは取得済み の扱いに
    • （スーパーマリオ3Dワールドと違って）このままいくとミスが確定（溶岩に落下など）な状況であってもやりなおし可能
  • 一方、コースをクリアせずにオプションから「コースをでる」を選択すると、ダイヤは取得していない扱いに
  • やりなおすと、おだいの達成／非達成の状態はリセット

コース開始方法の違いによる敵・ギミックの挙動への影響

  • いくつかのコースでは、自キャラを操作可能になる前に、「う～ん、まあ！」などの短いリアクションが発生
    • その中では、本のページからコースに入った場合は発生する一方で、ミスややりなおすでコースの最初に戻った場合は発生しないケースが多く存在
    • リアクションが発生している間も、敵やギミックは背後で動作
      • 特に　周期的な動きをする敵やギミックについて、本のページからコースに入った場合（画像左）と、ミスややりなおすでコースの最初に戻った場合（画像右）とで、自キャラが操作可能になる時点で初期位置が異なる という現象が発生（例：3-25（画像）のキラー）

強制スクロールコースでダイヤを取り逃がした場合

  • 取り逃したダイヤの位置によって、やりなおすを行うほうが早いかダイヤを2つ取得するコースで3つ取得するリカバリーをするほうが早いか、要判断
    • ダイヤを3つ取得するのにかかるタイム：2-05 < 3-14 < 2-17 < 3-20 < 2-14

RTA各コース解説

特に記載のないコースでは、ダイヤを3つすべて取得します 。ダイヤを2つ以下または全く取得しないコースの解説には、注意書き加えています。

■ 1-Prologue

ダッシュで最短距離を走りましょう。

  • 画像の赤丸のところで引っかかるので、多少膨らんだ走りにする必要あり
  • 画像の赤丸のところを越えたら青丸のあたりに向かって走り、そこでパワースターの方向に進路を変えるとよい

パワースター取得後のアニメーションは飛ばせません。ここに限らず、このゲームのいくつかのコースで発生するアニメーションは、コースのクリア実績がなければ飛ばすことができません。

■ 1-01：旅立ちのひっこぬき峠

GSN(2:56-) ／ SLDC(1:42-) を参照ください。

2つ目のダイヤを取得 → 次に引くプルスイッチ までの間のルートは 3秒程度ショートカットできるプレイ ができるが、難しいため、GSNもSLDCも挑戦していない

■ 1-02：キョロへいのロール神殿

コース開始直後に、画像左の矢印の方向にダッシュすることで、（水中ではなく）陸地の端に落下できます（水中に落下した場合は、やりなおす）。落下し始めたら、（必要ならカメラを回転と）右上の丸い回転岩を一時的にシビシビさせ、最速周期で向こう岸に渡れるようにしてください。

その後の動きは GSN(3:44-) ／ SLDC(2:31-) を参照ください。

引っかからずに丸い回転岩の穴に入るには、コインの真ん中を通るとよいです（画像右）。

このコース全体にわたって、上記「操作キャラの基礎動作」で述べた、シビシビ状態の継続について注意してください。

クリア後におだい達成のアニメーションが流れますが、おだいを達成しないよう行動するよりも時間のロスが少ないので、おだいを達成してかまいません。

■ 1-03：ふしぎなタッチストーン

全体の流れは SLDC(3:30-) を参照ください。以下、手順を記載します。

  1. 桃色の壁まで移動 → 桃色のブロックを左、水色のブロック上に → 水色ブロック内の1つ目のダイヤを取得 → わずかに折り返して桃色のブロックに乗り → 水色のブロックを下に
  2. 鍵のあるヒキヌケ草に向かって移動 → 水色のブロックに乗ったら、水色のブロックを上、桃色のブロックを右に → 鍵を引き抜き → 桃色のブロックの上へ投げ → 水色のブロックを下に → 桃色のブロックの穴に入り → 水色のブロックを上に → 折り返して2つ目のダイヤを取得
  3. 折り返して桃色のブロックの穴に入り → 水色のブロックを下に → 水色のブロックに乗り → 水色のブロックを上に → 鍵を取得 → 桃色のブロック下側にある 鍵の扉の真正面に向けて落下 → 落下中に反対方向に転換 （着地と同時に扉を開けるため） → 落下と同時に鍵の扉が開放 → 中にある3つ目のダイヤを取得 → やりなおす
  4. 水色のブロックを上に → 桃色のブロックの上に移動 → パワースターの方向に向かいつつ、水色のブロックを下、 桃色のブロックを3回移動 （やりなおし後において、おだいを達成しないようにするため）

■ 1-04：高原のキノコ岩

全体の流れは こちらのサンプル動画 を参照ください。以下、左上のブロックを動かすタイミングを記載します。

  1. コース開始時点：左上のブロック3つすべてを移動
  2. 2つ目のダイヤ取得後：右のブロックに乗り → 右のブロックを上に → 左のブロックの穴の中に入り → 左のブロックを下に
  3. 3つ目のダイヤ取得後：左のブロックの穴の中に入り → 左のブロックを上に → 真ん中のブロックに移動 → 左のブロックを下に → 真ん中のブロックを上に → 左のブロックに移動 → 左のブロックを上に

■ 1-05：ダブルチェリー宮殿

全体の流れは SLDC(5:06-) を参照ください（ただし、Pスイッチを踏んで折り返すところや、3つ目のダイヤを取得後にロスが発生しています）。

ダブルチェリーはかすめるように取り旋回すると、分裂した隊長と距離ができ、直後の2人同時Pスイッチ踏みがしやすくなります。

迷路の移動方法は画像のとおりです。

この後、左隊長が最上段から落下するタイミングと右隊長が2段目から落下するタイミングがあうように動かすとよいです。

3つ目のダイヤを取る際は、画像の状況からカメラ回転をせずに、右隊長をモグラがいる赤丸のあたりに向かって ひきぬくボタンを連打しながらダッシュさせます 。ひきぬいた後は、左隊長を奥まで進ませることを優先した後、右隊長にダイヤを回収させ赤色の地面まで戻ってこさせることを優先しながら左隊長をPスイッチまで進ませます。

■ 1-06：ヘイホーの街角

このコースには入りません。

■ 1-07：くるくる大図書館

まず、裏側の土管に入り逆さ側の地面で1つ目のダイヤを取得し、やりなおします。

やりなおし後は、以下の流れでクリアします。

  1. くるくるハンドルを反時計回りに90度 → 2つ目のダイヤ
  2. くるくるハンドルを180度（時計回り／反時計回りのどちらでも結果は同じ） → 3つ目のダイヤ
  3. くるくるハンドルを時計回り90度 → パワースター

注意点として、くるくるハンドルの近くでダッシュボタンを押す、もしくは、押していると、自キャラがくるくるハンドルをつかみ、ダッシュをボタンを離すまで動けなくなります。なので、 ダッシュはくるくるハンドルから少し離れたところでしてください 。ほかのくるくるハンドルが設置されたコースでも、同様に気をつけてください。

■ 1-08：投げまくりトロッコトンネル

強制スクロールコースのため、解説を省略します。

スタート地点から真右に進むと画像の赤線の柵に引っかかるので、やや右上→真右と移動する必要があります。

この先のコースでミスをする可能性がある場合は、ここでコインを100枚集めて1UPするのがよいです。ただし、おだいを達成しないよう、パタテンテンを最低1匹は残さなくてはなりません。左右に同時に現れるパタテンテンの片方を残すのがお勧めです。

■ 1-10：火ふき山の主

全体の流れは SLDC(9:07-) を参照ください。

最初は、画像左のようにダッシュしてください。そうすることで、ドラゴドンのブレスのうち画像右の赤丸の部分が壁に当たり、キノピオ隊長が壁から先に進んでいても被弾しません。

1つ目のダイヤを取得したら、そこの壁の裏に隠れて被弾を避けてください。

最初の階段を登っている途中で 動く壁が見えたら、シビシビさせて ください。シビシビは画像の位置あたりでZRボタンを離せば、壁の裏のはしごに間に合います。

慣れないうちは各所の壁の裏にとどまり被弾を避けたり、1度の被弾は覚悟で進んでもかまいません。ただし、1度被弾した場合は、後半で上から落ちてくる溶岩玉に気をつけてください。

■ 1-13：ジャングルのカブ大砲

SLDC(13:38-) を参照ください（ただし、誤ってモーキンにカブを当てており、ロスが発生しています）。

最初に落下する際は、画像のようにのびのびパックンに触れずに落ちるのが理想ですが、少し左に落ちるとトゲに被弾するので、そのリスクを考えると少し右に落ちてのびのびパックンを踏んでもよいです。

■ 1-14：荒野のニョキニョキ小屋

全体の流れは Elvaさんによるコース記録 を参照ください。

画像の場所のプルスイッチを引いて即座にその下の穴に入るには、 上下に動く黄色い床の端からダッシュでプルスイッチに近づきなるべく手前側でひきぬき、ダッシュを解除して折り返すとうまくいきやすい です。

コース記録と同等のタイムをだすには、画像（Elvaさんの上記動画から引用）の状態から奥側に動く黄色い床に逆らってダイヤを取ることが重要です。

なお、前述のプルスイッチを引き抜いて即座にその下に入れたが画像の動く黄色い床で1周期待った場合と、プルスイッチの下に入れなかったがそれ以降は最速で移動した場合とで、タイムはほぼ同じになります。

■ 1-15：疾走ラッカロード

GSN(17:09-) を参照ください。落下ミスしそうになったらやりなおすを行ってください。

おだいを達成しないために、最初の青いパタテンテンを踏んでいます。ここで踏むのに失敗した場合は、次の青いブロックの上のカブを引き抜き、その場のパタテンテンに当てるのがよいでしょう。

■ 1-16：ふしぎなドアのお化け屋敷

全体の流れは GSN(18:07-) を参照ください（ただし、最初のブロックを誤って多く移動させており、また、不要な被弾をしています）。

クリボータワーの上にある3つ目のダイヤの取り方ですが、クリボータワーの位置や誘導にかかる時間を勘案して、以下のどちらかのうち適切なほうを実施してください。

  1. クリボータワーがすでに壁に近い位置にいる、または、壁への誘導に時間がかからない場合：壁上のブロックを移動 → クリボータワーより1つ上の階層の扉に入り → 自動的にクリボータワーの上に落下
  2. クリボータワーが壁から離れた位置にいて、誘導に時間がかかる場合：クリボータワーと同じ階層に落下 → クリボータワーと同じ階層の扉に入り（このとき、クリボータワーが自然に誘導され） → 壁上の地面からクリボータワーの上にめがけて落下

GSNが成功している3つ目のダイヤだけを取りクリボータワーを踏まないテクニックは、難しめです。もしダイヤが取れないと大きなロスとなるので、安定をとってクリボータワーを確実に踏むようにしてもよいと思います。

全体の流れは こちらの動画 を参照ください（ただし、カメック戦でのはしご登りでもたついたのと、2つ目のダイヤを取るのに（はしごのサイドを落ちるのではなく）はしごを降りたことで、ロスが発生しています）。

関係知識

機種・バージョンによる主な違い

Speedrun.comでは機種により順位表が分かれています。

  • Wii U版（2014年11月13日発売）：
    • 「やりなおす」機能なし
      • RTA的にはSwitch版に比べて不利になるコースが存在（例：1-18）
    • キノピオのamiiboを使用すると、「かくれんぼ」が発動
      • 「100%」カテゴリーの挑戦に必須
    • おまけに収録されているコースは、スーパーマリオ3Dワールドをベースにしたコース
    • 一部の移動床が周期的に動作する（画像右）のではなく、GamePadに息を吹きかけることで動作（画像左）
      • RTA的にはSwitch版に比べて有利になるコースがいくつか存在（例：2-09（画像））

  • Switch版（2018年7月13日発売）：

    • 「かくれんぼ」がamiiboなしに可能
    • キノピオのamiiboを使用すると、使用したコースで無敵化（何度でも可能）
      • 詳細は、上記「RTA各コース解説」の「2-18：バトルタワーの頂上」を参照
    • おまけに収録されているコースは、スーパーマリオオデッセイをベースにしたコース
    • ウェディングスタイルのマリオ／ピーチ／クッパを使用すると、その時点からオデッセイのコースを遊べるように
      • 使用することで「All gems」・「100%」のカテゴリーのコース訪問順に影響がでるが、タイム短縮効果があるかは不明
    • バージョンによる主な違い：
      • Ver. 1.1.0：「2人で冒険」が実装
        • 「Switch (Co-Op)」カテゴリーがプレイ可能に
        • ネット越しでの2人プレイはできず、同じ場でプレイする必要あり
      • Ver. 1.2.0：「特別編」（DLC）対応、 「やりなおす」もこのバージョンから実装？（またはVer. 1.3.0から？）
        • 1-Prologue後～1-01までの間に発生するチュートリアルの内容に「VR」が含まれないため、このバージョンがRTA的には最速かも？
          • Speedrun.comでは バージョンの違いに関わらず順位表は一緒
      • Ver. 1.3.0：「VR」が実装

  • 3DS版（2018年7月13日発売）：

    • 未検証だが、Switch版とほぼ同様の模様。ただし、Switch版のVer. 1.1.0以降の機能は未提供。

このほかの違いについては、speedrun.comに英語で情報があります（ その1 、 その2 ）。

特別編（DLC）

  • 特別編を購入しているか否かで、1-Prologue終了～1-01開始までの間に発生するチュートリアルの内容に違いが発生。具体的には、 特別編を購入することでチュートリアルの内容が4秒短縮

    • SCRB_0 さんによる検証へのご協力に感謝

  • 特別編（英語名："Special Episode"）のコースをクリアーするカテゴリーも存在

表示言語の違いによるRTAの有利・不利

メッセージはボックス単位で一括表示され、現れるボックス数も各言語で同じと見受けられるため、表示言語による違いによるRTAの有利・不利はないと思います。
